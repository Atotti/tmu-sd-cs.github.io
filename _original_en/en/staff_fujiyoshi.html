Masaaki Fujiyoshi Laboratory

Associate Professor Masaaki Fujiyoshi

Media-aware Security / Image Processing / Value-added Media Processing

We generate, exchange, and use media information such as images every day, but media information are not always protected. So, I am interested in security technologies for media information, such as visual encryption that hides the content of photos, tamper detection against malicious editing of images, and so on. Based on such technologies, I am also engaged in research to add value to media information, research on security technologies using media information, and research on educational engineering.
